# The Weather Machine

A single-file static site visualizing the Weather Machine project's method, current
state, and roadmap. No framework, no build step — `index.html` ships as-is.

State lives in one `STATE` object at the foot of the `<script>` in `index.html` — the
hand-edit seam today, and the target a local sanitizer will emit into later.

No P&L is claimed or implied — there is none yet, by design.
