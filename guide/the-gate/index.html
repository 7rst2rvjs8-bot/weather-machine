Field guide · the basics

The 4/4 gate.

Before a machine built to trade is allowed near real money, one thing has to be settled: that it does exactly what it claims, every time, with no glitches. The gate is that check.

It verifies the boring, essential things — every trade recorded correctly, prices arriving clean and on time, the software's actions matching its own instructions, nothing breaking silently. The check has to come back clean for four separate trading sessions. “4/4” is four clean sessions out of four.

The part that's easy to miss

Passing the gate proves the machine is trustworthy. It does not prove the machine makes money. Those are two different claims, and keeping them apart is the whole discipline.

A measuring instrument has to be accurate before its readings mean anything. The gate certifies the instrument. Whether there is money to be made — the edge — is a separate question, answered later and forward, not here.

Go deeper — the technical version

The gate certifies the harness — execution, logging, and test scaffolding — across consecutive clean, unattended sessions against criteria fixed in advance, so the bar cannot move to fit the result. It explicitly does not certify the edge: a faithful instrument can still measure an edge that is not there. See The Machine.
