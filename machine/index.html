The machine

An instrument first.

Most of the build so far establishes that the machinery — execution, logging, self-checks — is trustworthy, held strictly apart from whether an edge exists. A faithful instrument can still measure an edge that is not there. Conflating the two is the failure this project is built to avoid.

02 / What's built

A book of strategies, most of it ahead.

ALLOCATOR
Spreads capital across strategies and prices in the luck of having chosen them. The destination, not today's reality.
STRATEGY #1
A short-dated momentum approach on options. Holds up in backtest; has never traded live. The deliberate baseline.
SHADOW STREAMS
Run live during market hours, logging every signal that would have fired — accumulating forward evidence before a dollar is risked. What this means →
UNIVERSE
9 liquid names — broad-market funds and a few high-volume single names. Tickers and signal logic are kept private.
Go deeper — how funding stays honest

The allocator carries a deflation over the number of strategies it ever chooses among, so the best-of-many luck measured at the single-strategy level cannot reappear unpriced at the portfolio level. Watching ideas is free; funding them passes a strict, pre-committed gate. See the search rule.
