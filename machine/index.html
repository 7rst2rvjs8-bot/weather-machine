The machine

A trustworthy instrument first.

Most of the work so far has gone into proving the machinery — the part that places trades, records them, and checks itself — is reliable, kept entirely separate from whether there’s actually money to be made. A perfectly accurate scale can still weigh something worthless. Confusing those two is the mistake this project is built to avoid.

Go deeper — how funding decisions stay honest

The allocator carries a penalty for the number of strategies it ever chooses among, so the “best of many” luck that’s measured at the single-strategy level can’t quietly reappear at the portfolio level. Watching ideas (free) is kept separate from funding them (a strict, pre-committed gate). See the search rule.
